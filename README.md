# jquery-ellipsis

Support multiple lines ellipsis

[http://plugins.jquery.com/ellipsis/](http://plugins.jquery.com/ellipsis/)

## Usage

Fit one line

```
$('#target').ellipsis();
```

Fit on two lines in the case of two or more lines

```
$('#target').ellipsis({
    row: 2
});
```

Change ellipsis character

```
$('#target').ellipsis({
    row: 2,
    char: '**'
});
```
